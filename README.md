# Diagonal Brick Web

Unity版と同じルールで遊べる、ブラウザ向けの7×7パズル版です。

## 起動方法

`WebGame/index.html` をブラウザで開いてください。

ローカルサーバーで確認する場合は、プロジェクト直下で以下を実行してから `http://localhost:8080/WebGame/` を開けます。

```powershell
python -m http.server 8080
```

## 実装内容

- 7×7盤面
- 3ブロックのランダム手札
- マウス・タッチのドラッグ配置
- 回転なし
- 縦・横・2方向の主対角線消去
- 配置 +10、ライン消去 1本/100・2本/250・3本/500・4本以上/1000
- 残り手札を置き切れるかを探索するゲームオーバー判定
- Retry / Title 画面
